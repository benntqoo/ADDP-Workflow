## 🔄 交互流程设计

### 启动流程
```
用户操作流程:
1. 打开应用
   ↓
2. 查看最近项目 / 创建新项目
   ↓
3. [点击项目卡片] → 自动填充配置
   或 [新建项目] → 手动配置
   ↓
4. 选择AI模型 (Claude/Gemini/Codex)
   ↓
5. 选择运行模式 (普通/YOLO)
   ↓
6. 点击 [🚀 启动AI工具]
   ↓
7. 确认对话框 → [确认] / [取消]
   ↓
8. 后台启动 + 状态反馈
   ↓
9. 成功提示 + 刷新最近项目
```
